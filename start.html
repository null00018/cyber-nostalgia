📡 ブログテンプレートをはじめよう

このページでは、あなた自身のブログを1クリックで始める方法を紹介します。

ステップ 1: GitHub アカウントを作成

GitHubに登録しておきましょう（無料）。

ステップ 2: テンプレートをコピー

下のボタンを押すと、このブログテンプレートを自分のGitHubにコピーできます。

🚀 Use This Template

ステップ 3: GitHub Pagesを有効に

  1. コピーされたリポジトリに移動
  2. [Settings] → [Pages] を選択
  3. "Branch: main / (root)" を選んで保存

数十秒後に、あなたのブログが公開されます。

ステップ 4: 投稿してみよう

URLの末尾に /submit.html を付けてアクセス → 投稿フォームが使えます。
