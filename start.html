📡 ブログテンプレートをはじめよう

このページでは、あなた自身のブログを GitHub 上だけで作り、記事投稿もワンクリックでできる方法を紹介します。

🛠 ステップ 1: GitHub アカウントを作成

  1. GitHub サインアップページにアクセス
  2. ユーザー名・メールアドレス・パスワードを入力
  3. ロボットではないことを確認し、[Create Account] をクリック
  4. 無料プラン（Free）を選んで次へ

📦 ステップ 2: テンプレートをコピー

下のボタンからこのテンプレートを自分のアカウントにコピー（＝リポジトリを作成）します。

🚀 Use This Template

※「リポジトリ」とはブログの中身を保存するフォルダのようなものです。

🌍 ステップ 3: GitHub Pages を有効に

  1. 作成した自分のリポジトリにアクセス
  2. 右上の「Settings」 → 左側の「Pages」へ
  3. "Branch: main / (root)" を選んで「Save」

約30秒〜1分後に、あなた専用のブログが公開されます。

✍️ ステップ 4: 投稿ページを設定して投稿してみよう

  1. まず、リポジトリ内の submit.html を安全のためにリネームします。
    例：submit-nf29ax.html など、自分だけがわかるランダムな名前に。
  2. その新しい名前を使って、あなたのブログURLの末尾に追加すれば投稿ページにアクセスできます。
    例：https://yourname.github.io/myblog/submit-nf29ax.html
  3. フォームが表示されたら、タイトル・本文・日付（未入力なら今日）を記入し、「投稿する」をクリック。
  4. 数秒後、投稿が自動的にHTML記事として生成され、トップページ（index.html）に自動反映されます。

💡 投稿内容は GitHub Issue 経由で処理され、ファイルとして生成されます。コードを書く必要はありません。
