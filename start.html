📡 ブログテンプレートをはじめよう

このページでは、あなた自身のブログを GitHub 上だけで作り、記事投稿もワンクリックでできる方法を紹介します。

🛠 ステップ 1: GitHub アカウントを作成

  1. GitHub サインアップページにアクセス
  2. ユーザー名・メールアドレス・パスワードを入力
  3. ロボットではないことを確認し、[Create Account] をクリック
  4. 無料プランのまま次へ

📦 ステップ 2: テンプレートをコピー

下のボタンを押すと、あなた専用のブログフォルダが作成されます。

🚀 Use This Template

ボタンを押すと次のような画面になります：

  1. Repository name：好きなブログ名を入力
  2. Description：空欄でOK
  3. Visibility："Public" のまま
  4. Create repository ボタンをクリック

これでテンプレートが作成されました。

🌍 ステップ 3: GitHub Pages を有効に

  1. 作成したリポジトリの上部メニューから Settings をクリック
  2. 左側のメニューから Pages を選択
  3. Branch のプルダウンを main / (root) に設定
  4. Save ボタンを忘れずにクリック

この設定をしないと、ブログは公開されません。

✍️ ステップ 4: 投稿ページを設定

  1. submit.html を Code のタブから開き、以下の部分を依須の情報に書き換え： 
    const username = "YOUR_GITHUB_USERNAME";
const repository = "YOUR_REPOSITORY_NAME";
    例: tanaka / my-blog
  2. 保存したら submit.html を自分の好きな名前にリネーム（例: submit-nf29ax.html など）
  3. そのURLにアクセス（例: https://yourname.github.io/myblog/submit-nf29ax.html）
  4. フォームで投稿したら、自動的に記事が作成され、index.html に反映されます

💡 投稿は GitHub Issue 経由で処理されます。
